## SI ALGO FALLA

### WiFi cae:
→ Cuento hasta 30, si no vuelve → PLAN-B
→ Nadie pierde puntos, continuamos sin pantalla

### QR no funciona:
→ Diles la URL: `localhost:8000` (si están en red local)
→ Si no: PLAN-B inmediatamente

### Navegador se congela:
→ Ctrl+Shift+Delete (borra caché)
→ Recarga página (F5)
→ Si sigue: PLAN-B

### Certificado no abre:
→ No importa, el certificado teórico ya se vio
→ Descargan luego, o les das certificado manual
